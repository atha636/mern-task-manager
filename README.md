# MERN Task Manager

Full Stack MERN Task Management Application.

## Features

- User Registration
- Login Authentication
- JWT Authorization
- Protected Routes
- Create Tasks
- View Own Tasks
- Update Tasks
- Delete Tasks

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- Tailwind CSS
- JWT
- bcryptjs

## Setup

Backend:

```bash
cd backend
npm install
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
